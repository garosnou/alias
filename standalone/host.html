Правила игры

Как играть:

  • Объясняйте слова своей команде за отведенное время
  • Нельзя использовать однокоренные слова
  • Нельзя показывать жестами или рисовать
  • Нажимайте "Правильно" если слово угадано
  • Нажимайте "Пропустить" если слово сложное

Примеры объяснений:

  • КОШКА - "Домашнее животное, которое мурлычет"
  • ТЕЛЕФОН - "Устройство для звонков"
  • ПОГОДА - "То, что показывает термометр"

Подсчет очков:

  • Правильный ответ: +1 очко
  • Пропущенное слово: 0 очков

Гибкий турнир

Создайте команды заранее или добавляйте по ходу. Состав и очки сохраняются в браузере до сброса. В каждом раунде выберите до 16 команд и нажмите «Вызвать команды» — зал увидит состав. Число кругов по умолчанию равно числу игроков у самой большой из выбранных команд (можно задать вручную ниже). У команды меньше игроков — те же игроки снова по кругу.

Только для кнопки «Быстро добавить» — не связано с числом кругов.
