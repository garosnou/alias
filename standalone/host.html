Игра в слова

Информация

Настройки игры

Встроенные слова — категории из списка ниже.

Пользовательские слова: один или несколько файлов .txt (по одному слову в строке). Все файлы объединяются в один пакет.

По темам: каждый файл .txt — отдельная тема. Перед игрой выберите тему из загруженных.

«Загрузить» — заменить все пакеты и сбросить прогресс по словам. «Добавить» — дописать файлы к уже загруженным, прогресс сохраняется.

    Пакеты не загружены

    Каждый файл — отдельная тема (название берётся из имени файла). «Загрузить» заменяет все темы и сбрасывает прогресс. «Добавить» — дописывает новые темы.

      Темы не загружены

      Положите картинки в папку bg рядом с файлами игры. Затем нажмите «Обновить фоны» и выделите все файлы в этой папке (Ctrl+A). Работает и при запуске с диска, без интернета.

      Полоса внизу игрового экрана зала: на всю ширину, без отступов по бокам и снизу; высота подстраивается под пропорции картинки. Рекомендуемый размер файла — до 1 МБ.

      Выберите тему

      Нажмите на тему, чтобы начать игру.

      Темы не загружены. Добавьте файлы .txt в настройках.

      Правила игры

      Как играть:

      • Объясняйте слова своей команде за отведенное время
      • Нельзя использовать однокоренные слова
      • Нельзя показывать жестами или рисовать
      • Нажимайте "Правильно" если слово угадано
      • Нажимайте "Пропустить" если слово сложное

      Примеры объяснений:

      • КОШКА - "Домашнее животное, которое мурлычет"
      • ТЕЛЕФОН - "Устройство для звонков"
      • ПОГОДА - "То, что показывает термометр"

      Источник слов

      • Встроенные — категории слов из настроек
      • Пользовательские — свои файлы, все объединяются в один пакет
      • По темам — каждый файл — отдельная тема; перед игрой выбираете тему

      Одиночная игра и игра на двоих

      • Одиночная игра — один раунд по времени из настроек
      • Игра на двоих — два раунда подряд: сначала один объясняет, затем меняются ролями; общий счёт — сумма двух раундов

      Подсчет очков:

      • Правильный ответ: +1 очко
      • Пропущенное слово: 0 очков

      Слова закончились

      Игра на паузе. Выберите один или несколько файлов .txt (по одному слову в строке) и нажмите «Загрузить и продолжить» — раунд продолжится с новым пакетом.

      Новые файлы заменят все пакеты в памяти и сбросит отметки «уже сыграно» для этих слов.

      Гибкий турнир

      Создайте команды заранее или добавляйте по ходу. Состав и очки сохраняются в браузере до сброса. В списке команд отметьте «Играет» у участников следующего раунда (до 16) и нажмите «Вызвать команды» — зал увидит состав. Число кругов по умолчанию равно числу игроков у самой большой из выбранных команд (можно задать вручную ниже). У команды меньше игроков — те же игроки снова по кругу.

      Только для кнопки «Быстро добавить» — не связано с числом кругов.
